# 6+ Poker Engine (C++)

## Description
This project is a C++ implementation of a 6+ (Short Deck) Poker engine developed as a personal project to practice Data Structures, Algorithms, and game logic design.

The engine simulates a player vs computer poker game with betting mechanics, card handling, and hand evaluation logic.

## Features
- Full deck and card system (Suit & Rank enums)
- Player vs Computer gameplay
- Hidden and visible card logic
- Hand value calculation
- War/tie system
- Object-Oriented design (Card, Deck, Player, Game classes)

## Technologies
- C++
- STL (vector, algorithm, etc.)
- Object-Oriented Programming
